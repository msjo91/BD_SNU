# Numpy
### ndarray
axis = 0 -> column  
axis = 1 -> row  
Axes: (Axis plural) Dimensions in numpy.ndarray object  
Rank: Number of axes

```
[3, 6]
# Rank 1 (1 axis)
# Length of axis: 2
# Shape: 1X2

[[1, 0, 2],
 [3, 5, 2]]
# Rank 2 (2 axes)
# Length of axis: 3
# Shape: 2X3

[[0, 0, 1],
 [1, 0, 2],
 [3, 5, 2]]
# Rank 3 (3 axes)
# Length of axis: 3
# Shape: 3X3
```
